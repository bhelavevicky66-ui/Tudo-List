# Todo App (Next.js + JavaScript)

A modern, aesthetically pleasing Todo application built with Next.js and React. This project features a premium "Glassmorphism" UI with smooth animations, implemented entirely with inline styles and JavaScript logic.

## üåü Features

- **Modern UI/UX**: Beautiful gradient backgrounds with glass-morphism cards.
- **Task Management**: Easily add, toggle, and delete tasks.
- **Filtering**: Segmented control to view All, Active, or Completed tasks.
- **Responsive**: Fully responsive design for all devices.
- **Pure JavaScript**: Built with standard JavaScript and inline CSS for zero build configuration issues.

## üöÄ Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/bhelavevicky66-ui/Tudo-List.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Tudo-List
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  **Open the app:**
    Visit [http://localhost:3000](http://localhost:3000) in your browser.
